## Adding new articles

1. Create md file in `wiki/<topic>/<post>.md`
    - Keep in mind, whatever directory structure you make here will be the link of the post!

2. Setup frontmatter (special markdown header) at the top of the file like so:

```
---
title: 'My First Post!'
path: '/<topic>/<post>.md'
---
